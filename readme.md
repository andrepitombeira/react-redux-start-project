# React Redux Start Project

### Includes

  * React
  * Redux
  * Webpack
  * Eslint
  * Babel
  * Karma
  * Enzyme

### Development setup

Install `yarn` as a dependency management for a faster and secure installation

    brew install yarn

Install dependencies with `yarn`

    yarn install

### Scripts:

Dev server

    yarn dev

Lint

    yarn lint

Test

    yarn test

Test with debug

    yarn test:debug
